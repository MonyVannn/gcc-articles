（金邊訊）柬埔寨政府決定延長西哈努克省投資優惠政策，協助當地工商業復甦。
<br/>
2024年1月底，政府啟動了《2024年西哈努克省投資促進特別計畫》，旨在吸引國內外企業投資西哈努克省，推動當地經濟成長並促進工業和商業的發展。
<br/>
根據財經部最新發布的公告，政府決定將此政策延長至2025年底。
<br/>
1月23日，財經部國務秘書兼西哈努克省投資促進工作組組長韓沙賀主持召開第13次工作組會議，討論了投資者申請享受特殊優惠的相關事項。
<br/>
會議原則上批准了42個項目享有投資優惠，總投資額達5.64億美元。其中，包括15個與「爛尾樓」相關的項目、12個新建項目、3個擴產項目，12個不涉及「爛尾樓」項目。這些項目涉及海關倉庫、高龍撒冷島住宅項目、生態農業旅遊度假村項目、工業園區項目、酒店項目和多功能辦公大樓項目等，預計可創造7308個就業機會。
<br/>
韓沙賀指出，政府決定繼續實施《2024年西哈努克省投資促進特別計畫》至2025年底，並將延續所有現有的激勵措施，以進一步推動西哈努克省的經濟發展，助力實現政府的長遠發展目標。
<br/>
他強調，工作小組必須持續履行三項核心任務，一是研究並提供更多的特殊激勵措施；二是吸引並促進更多投資；三是推動當地經濟的永續發展。
<br/>
值得一提的是，截至2024年12月中旬，西哈努克省共批准了213個享有特殊優惠的項目，總投資額達60.26億美元。在這些項目中，涉及「爛尾樓」的項目有103個，投資額達25.1億美元；不涉及「爛尾樓」的項目有86個，投資額為26.99億美元；此外，還有17個擴產項目，投資金額為7.68億美元。
<br/>
政府將西哈努克省視為繼首都金邊之後的第二大“經濟極”，並計劃將該省發展成為“多元用途經濟特區”，以推動柬埔寨經濟實現可持續發展。
<br/>
為充分挖掘西哈努克省的巨大潛力，並振興因新冠疫情受創的西港產業和旅遊業，政府決定推出這一特別項目，透過提供稅務和非稅務激勵措施，吸引更多投資，進一步推動西省的全面發展。
